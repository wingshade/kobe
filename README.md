## 包車 vs 大眾交通比較

| 方案 | 費用 (6人) | 優點 | 缺點 |
|------|------------|------|------|
| **大眾交通** | 約 ¥48,720 (¥8,120/人) | 便宜、班次多 | 需要換乘、帶小孩較累 |
| **包車一日** | 約 ¥50,000-70,000/天 | 輕鬆、彈性 | 費用較高 |
| **包車姬路城一日** | 約 ¥60,000 | 姬路最遠，包車最輕鬆 | 其他天用大眾交通即可 |

### 建議方案
**大眾交通為主**。因為：
1. 神戶市區交通非常方便，Port Liner、JR、地鐵都很便捷
2. 景點到車站的距離都在步行範圍內
3. 包車費用約為大眾交通的 3-4 倍
4. 如果小孩容易累，可考慮**只在姬路城那天包車**（車程最長）

- **神戶動物王國套票**：含 Port Liner 來回+門票，官網或車站購買

- **目前票價（2025年）**：成人 ¥1,000 / 兒童 ¥300
- **姬路城＋好古園套票**：成人 ¥1,050 / 兒童 ¥360
- **開放時間**：
  - 夏季（4/29-8/31）：9:00-17:00
  - 其他時間：9:00-16:00

⚠️ **重要提醒**：姬路市政府已宣布自 **2026年3月1日** 起實施「雙重價格」制度，非姬路市民門票將從 ¥1,000 調漲至 **¥2,500**（漲幅 2.5 倍）。本次行程（2025/12）仍適用現行票價。

### 神戶港 Harborland 特色
**主要景點：**
- **umie MOSAIC 購物中心**：北館、南館、馬賽克廣場共 230+ 家店舖
- **MOSAIC 摩天輪**：港區地標，車廂有冷氣，夜間點燈很美
- **神戶港塔**：108公尺高，2024年重新開幕，4樓以下免費參觀
- **跳ね橋（可動橋）**：日本首座行人專用可動橋，夜間點亮 1,700 盞燈飾
- **麵包超人博物館購物商場**：1樓免費入場

**從飯店前往方式：**
- JR 三宮站 → JR 神戶站（1站，約3分鐘，¥130）→ 步行5分鐘
- 從 JR 神戶站經地下街 DUO KOBE 可直通 umie
